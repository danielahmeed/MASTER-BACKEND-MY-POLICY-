================================================================================
HOW TO CONVERT INGESTION_MODULE_CODE_EXPLAINED_DETAILED.md TO PDF
================================================================================

OPTION 1: Using Python script (md_to_pdf.py)
---------------------------------------------
1. Open a terminal in this Documentation folder.

2. Install required packages:
   pip install markdown xhtml2pdf

3. Run the script:
   python md_to_pdf.py

4. Output files created:
   - INGESTION_MODULE_CODE_EXPLAINED_DETAILED.html (intermediate)
   - INGESTION_MODULE_CODE_EXPLAINED_DETAILED.pdf  (final)

OPTION 2: Using your browser (manual)
-------------------------------------
1. Open INGESTION_MODULE_CODE_EXPLAINED_DETAILED.md in a Markdown viewer
   (e.g., VS Code, GitHub, or a Markdown extension).

2. Or convert MD to HTML first using any online tool like:
   - https://markdowntohtml.com/
   - Paste the content and download HTML.

3. Open the HTML file in Chrome or Edge.

4. Press Ctrl+P (Print) and choose "Save as PDF" as the destination.

OPTION 3: Using pandoc (if installed)
-------------------------------------
pandoc INGESTION_MODULE_CODE_EXPLAINED_DETAILED.md -o INGESTION_MODULE_CODE_EXPLAINED_DETAILED.pdf
